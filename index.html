Penny Lane Properties

Built over a four-day sprint, Penny Lane Properties is an estate agency listing site. We used PHP and SQL to create a database and app using data from an API. The project used OOP to create a codebase with maintainable architecture and our focus was on writing code that adhered to SOLID principles. The project was fully unit tested. During the sprint I focused on writing and implementing SQL queries and display functions as well as the data validation for the form input.

Food.Delivered

Food.Delivered is a React app built based on a client-supplied design with data fetched from an existing API. We front-loaded all our app architecture to design a component structure that made use of shared state, allowing a seamless user experience. My focus in this project was on developing the order card and basket totals logic.

Gopher Glory

Gopher Glory is web game where players have to catch all the gophers before they dig up all their carrots. The game is built with Javascript and has a high-quality design. The user can select the size of their playing board as well as how many gophers have hidden themselves in their carrot patch!
